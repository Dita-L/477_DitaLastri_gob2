Membuat Models
    >>User
        >>ID
        >>Username      Validasi uniqueindex, tidak boleh kosong, harus terisi
        >>Email         Validasi format email, uniqueindex, tidak boleh kosong, harus terisi
        >>Password      Validasi tidak  boleh kosong, minimal 6 karakter
        >>Age           Validasi tidak boleh kosong, minimal umur diatas 8
        >>Created_At    Date
        >>Updated_At    Date

Membuat User Controllers
    >>POST (/users/register) == Create
        request body:
            {
                "age": int,
                "email": string,
                "password": string,
                "username": string
            }
        
        response:
            {
                "age": int,
                "email": string,
                "id": int,
                "username": string
            }
        
        Password di hash dengan bcrypt sebelum disimpan ke database
            >>buat seperti jwt, ada helpers
    
    >>POST (/users/login) 
        request body :
            {
                "email": string,
                "password": string
            }
        
        response: //membuat token
            {
                "token": string
            }
        
        Logika: cek email dan password
            >> cek password juga menggunakan bcrypt
            >> helpers, bcrypt, ComparePass

    >>PUT (/users) == Update
        request
            >>headers: Autorisasi (token)
            >>body:
                {
                    "email": "string",
                    "username": "string"
                }

        response
            {
                "id": int,
                "email": string,
                "username": string,
                "age": int,
                "updated_at": date
            }

        Melewati Autentikasi dulu
            >>verivikasi token

    >>DELETE (/users)
        request
            >>headers: Autorisasi (token)
        
        response
