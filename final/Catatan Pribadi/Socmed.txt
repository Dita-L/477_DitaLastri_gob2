Membuat Models
    >>Socmed
        >>ID
        >>Name          String/vachar       Validasi tidak boleh kosong
        >>Socmed_URL    String/text         Validasi tidak boleh kosong
        >>UserID
        >>Created_At    Date
        >>Updated_At    Date

Membuat Controller
    >>POST (/socialmedias)
        >>Authorization (token)
        >> request body:
            {
                "name": "string",
                "social_media_url": "string"
            }
        
        >>response 
            {
                "id": interger,
                "name": "string",
                "social_media_url": "string",
                "user_id": "interger",
                "created_at": "date"
            }
        >> endpoint ini melalui autentikasi dan autorisasi
            >>autentikasi (token)
    
    >>GET (/socialmedias)
        >>Authorization (token)

        >>response
            {
                "social_medias": [
                    "id": interger,
                    "name": "string",
                    "social_media_url": "string",
                    "UserId": "interger",
                    "createdAt": "date",
                    "updatedAt": "date",
                    "User": {
                        "id": interger,
                        "username": "string",
                        "profile_image_url": "string"
                    }
                ]
            }

        >>Endpoint ini perlu melewati Autentikasi dan Autorisasi terlebih dahulu.
            >>Autentikasi (token)
    
    >>PUT (/socialmedias/:socialMediaId)
        >>Authorization (token)
        >>params : socialMediaId (int)
        >>request body:
            {
                "name": "string",
                "social_media_url": "string"
            }

        >>response
            {
                "id": interger,
                "name": "string",
                "social_media_url": "string",
                "user_id": interger,
                "updated_at": "date"
            }
        
        >>Endpoint ini perlu melewati Autentikasi dan Autorisasi terlebih dahulu.
            >>Autentikasi (token)
            >>Autorisasi : User hanya boleh meng-update data socmednya sendiri
    
    >>DELETE (/socialmedias/:socialMediaId)
        >>Authorization (token)
        >>params: socialMediaId

        >>response:
            {
                "message": "Your social media has been successfully deleted"
            }
        
        >>Endpoint ini perlu melewati Autentikasi dan Autorisasi terlebih dahulu.
            >>Autentikasi (token)
            >>Autorisasi : User hanya boleh menghapus data socmednya sendiri
